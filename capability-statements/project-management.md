# Capability Statement — Project Management

## Overview

AJ Digital provides project planning, coordination, and delivery support for client initiatives — marketing launches, systems implementations, workflow rollouts, and event productions. Led by Tyrone Nelms (see `bios/consultant-bio.md`), Google-certified in project management with hands-on delivery experience in ClickUp, Asana, and Notion.

## Ideal Client

Founder-led businesses running real projects without project infrastructure — initiatives tracked in chat threads, deadlines living in one person's calendar, and no shared view of status.

## Problems Solved

- Projects with no single owner of the plan, timeline, or status.
- Stakeholders who find out about slips after they happen.
- Work tracked across chat, email, and memory instead of one system.
- Recurring project types (launches, events, onboardings) rebuilt from scratch every time.

## Approach

1. **Discovery and scoping** — translate the business goal into deliverables, owners, and a timeline.
2. **System setup** — stand up the project in the team's tool (ClickUp/Asana/Notion) with templates that outlive the engagement.
3. **Cadence** — status reporting and stakeholder communication on a fixed rhythm.
4. **Close-out** — documentation of what shipped and a reusable template for the next run.
