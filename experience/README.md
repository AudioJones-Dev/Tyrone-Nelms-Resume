## Conventions

- New records start from [`templates/experience-template.md`](../templates/experience-template.md).
- Keep source-variant notes (e.g. conflicting date ranges across old resumes) in these files rather than deleting them.
- Chronology conflicts are resolved in [`docs/career-history.md`](../docs/career-history.md), which is canonical.
- Status labels: `verified`, `needs_verification`, `current_context`, `private_context` (see [CONTRIBUTING](../CONTRIBUTING.md)).
